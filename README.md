# python-book-examples

## Environment
### Python Version
```bash
python 3.8.1
```

### Setup Environment
- Uses `pyenv` and `venv`.
```bash
# Make sure that pyenv is installed.
# To install 3.8-dev (2020.1.2), 2.12 or higer is required.
$ pyenv install 3.8-dev

# Create virtualenv using venv.
$ python3 -m venv PATH_TO_CREATE_VENV
```
